# Python File Organizer

This project is a simple Python script that automatically organizes files
in a folder into subfolders based on file type (images, documents, videos, etc.).

## How It Works
- Scans a target folder
- Detects file extensions
- Moves files into categorized folders

## Technologies Used
- Python
- os module
- shutil module

## Use Case
This script can help users clean up messy download folders
and save time by automating file organization.
